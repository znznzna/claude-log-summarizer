# Claude Log Summarizer - Obsidian Plugin

## プロジェクト概要

ClaudeCodeの会話ログを自動的に要約し、Obsidian Daily Noteに作業日報として挿入するプラグイン。

## 技術スタック

- **言語**: TypeScript
- **ビルドツール**: esbuild
- **API**: GLM-4-Flash (Z.ai)
- **対象**: Obsidian v1.0.0+

## プロジェクト構成

```
claude-log-summarizer/
├── main.ts           # プラグインのメインロジック
├── settings.ts       # 設定タブUI
├── types.ts          # TypeScript型定義
├── manifest.json     # プラグインメタデータ
├── package.json      # npm設定

```

## 実装機能

### コア機能
- ✅ ログファイル読み取り（`YYYY年MM月DD日.md`形式）
- ✅ GLM-4-Flash API統合（Z.ai経由）
- ✅ Daily Note書き込み（`YYYY-MM-DD.md`形式）
- ✅ 設定画面（APIキー、パス、見出しのカスタマイズ）
- ✅ コマンドパレット登録（「作業日報を生成」）
- ✅ リボンアイコン（サイドバーからワンクリック実行）
- ✅ 自動実行機能（Obsidian起動時に前日分を生成）

### 設定項目
- GLM APIキー
- ログファイルパス（デフォルト: `AI-Output/_CLAUDE/Talklog`）
- Daily Noteパス（デフォルト: `Daily`）
- 挿入先見出し（デフォルト: `## 作業日報`）
- 自動実行ON/OFF

## 使い方

### 初回設定
1. Obsidianを再起動
2. 設定 → コミュニティプラグイン → 「Claude Log Summarizer」を有効化
3. プラグイン設定でGLM APIキーを入力

### 実行方法
- **手動実行**: Ctrl/Cmd + P → 「作業日報を生成」
- **リボンアイコン**: 左サイドバーの📄アイコンをクリック
- **自動実行**: 設定で「起動時に自動実行」をONにする
